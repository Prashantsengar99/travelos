# 🌍 TravelOS — AI Travel Companion & Budget Planner

[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge)](https://travelos-neon.vercel.app/)
[![Backend API](https://img.shields.io/badge/Backend-API-blue?style=for-the-badge)](https://travelos-mkpn.onrender.com/api/health)

TravelOS ek AI-powered travel companion hai jo aapki trips plan karne, budget track karne, aur smart suggestions dene mein madad karta hai. 🎒✨

---

## ✨ Features

### 📊 Dashboard
- Real-time budget tracking
- Daily spending trends with charts
- Category-wise expense breakdown
- Budget health indicator
- Smart alerts and recommendations

### ✈️ Trip Management
- Create unlimited trips
- Set budget, dates, and travel type
- Track expenses per trip
- Active trip switching

### 💰 Expense Tracking
- Add expenses with categories
- 8+ categories (Food, Hotel, Transport, Shopping, etc.)
- Date-wise expense tracking
- Delete and manage expenses

### 📈 Analytics
- Budget vs Expense comparison
- Category distribution charts
- Daily spending trends
- Detailed cost breakdown

### 🤖 AI Advisor
- Smart budget suggestions
- Savings score
- Personalized tips based on spending
- Local travel recommendations

### 🌤️ Weather
- Real-time weather updates
- 5-day forecast
- City-wise weather search
- Packing tips based on weather

### 🧳 Packing List
- Pre-defined packing checklist
- Add custom items
- Progress tracking
- Save to cloud

### 👥 Split Expenses
- Group expense sharing
- Fair settlement calculation
- Balance tracking
- Payment history

### 💱 Currency Converter
- 10+ currencies
- Real-time conversion
- Quick reference rates

### 🏆 Badges
- Achievements system
- 8+ badges to unlock
- Gamification

### 💬 AI Chatbot
- 24/7 travel assistant
- Budget tips
- Food and place suggestions
- Weather queries

## 🛠️ Tech Stack

### Frontend
- **HTML5** - Structure
- **CSS3** - Custom styling with dark theme
- **JavaScript** - Vanilla JS (No framework)
- **Chart.js** - Data visualization
- **Tailwind CSS** - Utility-first CSS
- **Font Awesome** - Icons

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB Atlas** - Cloud database
- **JWT** - Authentication
- **bcryptjs** - Password hashing
- **CORS** - Cross-origin resource sharing

### Deployment
- **Render** - Backend hosting
- **Vercel** - Frontend hosting
- **MongoDB Atlas** - Database hosting
